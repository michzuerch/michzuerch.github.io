LocateIPAddress

Projekt auf Github

Java 10 Maven MongoDB Spring-Boot

Mit diesem Programm kann man eine Datenbank erstellen die sämtliche IP-Adressen enthält. Die Daten werden von der Firma MaxMind als Opensource zu Verfügung gestellt: Download

Es gibt eine Uploadfunktion welche die ca. 38 MB grosse Datei in eine MongoDB-Datenbank überträgt. Weiterhin kann man zu jedem IP-Range die entsprechende Stadt suchen.
